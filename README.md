# devops-lecture

A simple "Hello, Friends!" application written in Zig.

This project is configured with a `Makefile` for common tasks.

## Prerequisites

- [Zig](https://ziglang.org/)
- [Make](https://www.gnu.org/software/make/)

## Building and Running

### Using Make

- **Build:** `make build`
  - This will create the executable at `zig-out/bin/devops-lecture`.
- **Run:** `make run`
  - This will build and run the application.
- **Test:** `make test`
  - This will run the tests.
- **Clean:** `make clean`
  - This will remove build artifacts.
